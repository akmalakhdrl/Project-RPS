Step 1: USER mengisi form di browser
   â†“
   User input data mata kuliah:
   - Nama, kode, semester, SKS
   - Deskripsi (opsional)
   - CPL, CPMK (opsional)
   - Metode, media (opsional)

Step 2: USER klik tombol "Generate RPS"
   â†“
   RPSForm.jsx:
   - Validasi data
   - Format data ke JSON
   - Set loading state = true

Step 3: FRONTEND kirim data ke BACKEND
   â†“
   api.js:
   - axios.post('http://localhost:5000/api/rps/generate', data)
   - Kirim via HTTP POST

Step 4: BACKEND terima request
   â†“
   server.js â†’ rpsRoutes.js â†’ rpsController.js
   - Parse JSON dari request body
   - Validasi data

Step 5: BACKEND panggil AI Service
   â†“
   aiService.js:
   - Buat prompt untuk OpenAI
   - Kirim request ke OpenAI API
   - Terima response (konten RPS)
   - Parse & format response

Step 6: BACKEND generate file JSON
   â†“
   rpsController.js:
   - Gabungkan data input + AI response
   - Save ke file JSON di folder outputs/
   - Generate filename: rps_[kode]_[timestamp].json

Step 7: BACKEND generate file DOCX
   â†“
   docxService.js:
   - Buat dokumen Word baru
   - Format heading, paragraf, tabel
   - Save ke file DOCX di folder outputs/
   - Generate filename: rps_[kode]_[timestamp].docx

Step 8: BACKEND kirim response ke FRONTEND
   â†“
   Response JSON:
   {
     success: true,
     message: "RPS berhasil digenerate",
     data: { ... data RPS lengkap ... },
     files: {
       json: "rps_IF101_1234567890.json",
       docx: "rps_IF101_1234567890.docx"
     }
   }

Step 9: FRONTEND terima response
   â†“
   api.js â†’ RPSForm.jsx â†’ App.jsx
   - Set loading = false
   - Update state dengan data hasil
   - Switch tampilan ke RPSResult

Step 10: USER lihat hasil
   â†“
   RPSResult.jsx:
   - Tampilkan preview RPS
   - Show download button JSON & DOCX

Step 11: USER klik download
   â†“
